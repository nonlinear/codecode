Add your own font

Basic

  1. Have your font file (find cool and free ones at dafont)
  2. Generate webfont at fontsquirrel
  3. Move resulting webfonts to

    app/fonts

  4. type

    font-face

    on your SASS file

    variable should match filename

Advanced

  1. Install Homebrew and dependencies:

    brew install fontforge ttfautohint ttf2eot batik

  2. Add a

    *.ttf

    or

    *.otf

    font on

    app/fontgen

  3. copy filename, rename if needed

    it will become font-family

  4. on terminal, inside your project folder, type

    gulp fontgen

  5. go to top of

    app/sass/layout.sass

    and type

    fontface

    , paste filename as variable

    add this declaration before you call font-family for it to work

  6. on

    app/sass/style.sass

    , use font filename as variable for either

    $font-text

    or

    $font-title
